# vue-component

> Roll your Vue components with Rollup.

Fork from [vue-plugin-simple](https://github.com/znck/vue-plugin-simple).

## 技术栈

* VueJs
* Rollup
* Eslint

## 安装

确保已经安装workplus-cli

```bash
$ npm install workplus-cli -g
```

然后执行以下命令：

```bash
$ workplus start vue-component my-project
```

安装完成后，进入项目目录，执行npm install
